# Movie Reviewer 🎬

A web app to search and display detailed information about any movie using the OMDB API.

## Features
- Search movies by title
- Displays rating, plot, cast, director, genre, runtime, and more
- Movie poster display
- Responsive design with mobile hamburger menu

## Tech Stack
HTML, CSS, Vanilla JavaScript, OMDB API

## Setup
1. Get a free API key from [omdbapi.com](https://www.omdbapi.com/apikey.aspx)
2. Replace the `API_KEY` value in `script.js`
3. Open `index.html` in your browser
